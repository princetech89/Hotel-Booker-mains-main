# Hotel Booker

This is a web application for booking hotels.

## Deployment on Vercel

To deploy this application on Vercel, follow these steps:

1.  **Install the Vercel CLI:**

    ```bash
    npm install -g vercel
    ```

2.  **Deploy the application:**

    Run the following command in the root of the project:

    ```bash
    vercel
    ```

3.  **Follow the on-screen instructions:**

    The Vercel CLI will guide you through the process of linking the project to your Vercel account and deploying it. You can use the default answers for all the questions.
